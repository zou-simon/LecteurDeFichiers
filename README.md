# Mini-projet : Lecteur de fichiers
 
Lecteur de fichiers (surtout aux fichiers texte) en Java.

- Lecture d'un fichier texte
- Lecture d'un fichier texte à l'envers
- Lecture d'un fichier texte de manière palindromique (en terme 
de caractères).
- Comparateur de fichiers textes
